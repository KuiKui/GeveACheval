## Dev

Installation de Bundler :
```
$ sudo gem install bundler
```

Installation de Jekyll :
```
$ bundle install
```

Clone du projet :
```
$ git clone https://github.com/KuiKui/GeveACheval.git
$ cd GeveACheval
```

Lancement du serveur de dev sur `http://0.0.0.0:4000` :
```
$ bundle exec jekyll serve -w
```
